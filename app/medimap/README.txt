실행방법

1. docker와 git 을 설치하고 실행합니다.

2. 터미널에 

git clone https://github.com/jinupark0708/MediMap.git
cd MediMap/app/medimap
docker-compose down -v
docker-compose up --build

그대로 입력합니다.

3. 브라우저에서 아래 주소로 접속합니다.
 http://localhost:8080/index.html

시작화면에서 버튼을 누르면 로그인 화면으로 갑니다
로그인화면에서 sign up을 통해 회원가입을 할수있고
로그인을 진행할 수 있습니다.

로그인에 성공하면 메인화면으로 갑니다

지도에 병원 마커와 현재 위치 마커가 뜨고 병원 마커를 클릭하면
병원의 이름과 주소, 보유 약품들과 그 수량이 표시됩니다.
지도를 이동하면 그에맞게 주변의 약국들이 업데이트 됩니다.

원래는 병원의 관리자들이 직접 기입하여야하는 정보들이지만
현재는 불가한 일이기 때문에 영남대 근처에 있는 약국들의 정보를 기입해두었습니다.

상단에 검색창 왼쪽 recommend 버튼을 클릭하면 증상을 기반으로 한 약을 추천받을 수 있고 주변에 그 약을 보유하고 있는 약국들이 출력됩니다.

검색창에 약 이름(타이레놀, 판콜에스, 디페린 등)을 입력하면 약의 사진, 효능, 용법, 주의사항과 그 약을 판매하고있는 약국들이 출력됩니다.

검색창 오른쪽의 member info 버튼을 클릭하면 비밀번호 변경, 회원 탈퇴를 진행할 수 있습니다.

그리고 로그인 시 customer 타입의 id가 아닌 manager 타입의 id로 로그인할 경우 오른쪽 상단에 manage inventory 버튼이 표시됩니다.
해당 버튼을 클릭할 시 각 약국에 물품을 추가,삭제 할 수 있고 수량 조절을 할 수 있습니다.
수정사항이 있을시 db가 업데이트 됩니다.

오른쪽 상단 logout버튼을 클릭하면 로그아웃 되고 로그인 화면으로 돌아갑니다.
